# Career Guidance Expert System

**Career Guidance Expert System** that recommends careers based on your interests, skills, education, work style, salary expectations, and experience.

## Features

- Multi-step form to collect user profile:
  - Interests
  - Skills
  - Education
  - Work style
  - Salary expectations
  - Experience
- Personalized career recommendations with match scores
- Insights and suggestions to improve career alignment
- Displays career description, growth prospects, and typical pathways
- Reset option to start a new assessment
- Responsive and user-friendly UI built with React

## Technologies Used

- **React** for frontend
- **Python** for backend logic and expert system reasoning 
- **Tailwind CSS** for styling
- **Lucide Icons** for UI icons
- JavaScript for frontend logic and match calculations
